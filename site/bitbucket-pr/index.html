Auth is a Bitbucket Cloud App Password in BB_USER / BB_TOKEN, with workspace and repo defaulting from the git remote.

Run it without a tab switch.

Set your App Password once, then drive the whole pull request from the chat. It is one of the Atlassian integration skills, so it pairs with jira-api on the same workflow.

  • Open, review, approve and merge through the Bitbucket Cloud API.
  • Drop inline review comments anchored to a file and line.
  • Merge with commit, squash, or fast-forward strategy.
