Open the catalog.

The catalog ships inside the plugin and is never auto-loaded, so nothing is active until you install it. Run the command, reply with the numbers or names you want, and pick where they land.

  • A numbered list with every skill's description and install status.
  • Install to this project only, or globally for every project.
  • An existing skill is never overwritten without a warning.
