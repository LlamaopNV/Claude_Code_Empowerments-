Browse a repo without cloning it.

Four read-only operations over the Bitbucket Cloud API. Auth comes from BB_USER and BB_TOKEN or your ~/.bitbucket file, and every call is a GET.

  • List repos in a workspace, or branches with a filter.
  • Read the commit log for a branch or a single path.
  • Open any file at a branch, tag, or commit hash.

Part of the Atlassian integration suite. All operations read-only.

install and invoke
# add the marketplace once
/plugin marketplace add LlamaopNV/Claude_Code_Empowerments-
> then /plugin, install bitbucket-repo
# read a file at a tag, a branch, or a commit
/bitbucket-repo file v2.3.0 src/index.ts
# list branches, or the commit log of a path
/bitbucket-repo branches release
/bitbucket-repo commits main src/index.ts 10
# or just ask
"show me src/index.ts as it was at v2.3.0"
"what changed on the release branch?"
