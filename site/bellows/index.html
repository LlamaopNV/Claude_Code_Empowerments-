The four surfaces

How air becomes heat

One continuous airflow: open the intake, gauge what is flowing, aim it at the fire, then duct it into your own shop. Every surface rides the same zero-dependency client, and no key material or prompt content ever touches disk.

/nim-setup · open the intake

Guided key onboarding, verified live

Walks you through a free build.nvidia.com account and an nvapi- key, verifies it against the real endpoint before persisting anything, and doubles as a health check forever after. Idempotent: run it any time to ask "is my air still flowing?"

/nim-models · gauge the airflow

An annotated live inventory

Pulls the actual model list from /v1/models and annotates it: which id is the coding default, which one reasons deliberately, which handles long context. Only models your account really has are ever shown.

/nim + nim_chat · pump it into the fire

Delegate the grunt-work, keep the judgment

One-shot a prompt from the command line, or let Claude reach for the auto-started MCP tools mid-task. The skill teaches it when offloading actually pays:

Bulk transforms. Send one exemplar and the rule, apply the pattern locally, verify with tests.
Second opinions. Describe the stuck bug to a reasoning model. Disagreement is signal.
Parallel drafts. Ask for three candidate implementations while writing the one you believe in.
Big-input summarization. Pipe in the logs too large to read inline, spot-check what you act on.
nim-integration-playbook · duct it into your shop

The instructor for your own apps

NVIDIA's endpoints are OpenAI-compatible, so your app uses the standard OpenAI SDK pointed at their base URL. The playbook covers JS/TS, Python, raw HTTP, streaming, model choice, and key hygiene, written native to your existing stack.

The honesty rule the plugin ships with

Delegated output is a draft, never an authority. Every answer is attributed to the model that produced it, verified before it is relayed, and the skill refuses tasks that need repo context the remote model cannot see. Prompts go to NVIDIA's cloud, so anything sensitive is asked about first.
